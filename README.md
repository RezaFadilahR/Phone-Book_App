# Phone-Book_App
Phone Book adalah aplikasi manajemen kontak yang sederhana dan efisien, dirancang untuk memberikan pengalaman pengguna yang intuitif. Dikembangkan dengan menggunakan teknologi React, Emotion, dan TypeScript


Fitur Utama:
Tambah Kontak: Tambahkan kontak baru dengan mudah melalui antarmuka yang ramah pengguna.
Hapus Kontak: Hapus kontak yang tidak diperlukan atau tidak aktif.
Pencarian Cepat: Temukan kontak dengan cepat berdasarkan nama atau nomor telepon.
Daftar Favorit: Tandai kontak favorit Anda untuk mengakses mereka dengan lebih cepat.
Teknologi yang Digunakan


Aplikasi ini dikembangkan menggunakan teknologi terkini, termasuk:

React: Sebagai basis pengembangan antarmuka pengguna yang responsif.
Emotion: Untuk manajemen gaya dan tata letak yang efisien.
TypeScript: Meningkatkan keamanan dan keterbacaan kode.
AOS: Menambahkan animasi halus untuk meningkatkan pengalaman pengguna.


Cara Menggunakan
Clone Repositori:
git clone https://github.com/username/repo.git

Install Dependensi:
cd repo
npm install


Jalankan Aplikasi:
npm start
